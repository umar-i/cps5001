# Data

Optional folder for simulation inputs and outputs.

Recommended:
- `data/scenarios/` for scenario/event inputs (CSV).
- `data/out/` for exported metrics (CSV) to support evaluation/visualisation in the report.

Included samples:
- `data/scenarios/mini-nodes.csv`
- `data/scenarios/mini-edges.csv`
- `data/scenarios/mini-events.csv`
